# 📊 Sales & Customer Revenue Analytics (SQL Project)

## 🧠 Overview
This project analyses retail sales data using SQL to uncover revenue trends, customer behaviour, and product performance.

---

## 🎯 Objectives
- Identify top-performing products
- Analyse customer lifetime value
- Track monthly revenue trends
- Segment customers by value

---

## 🛠️ Tools
- PostgreSQL (SQL)
- Excel (basic validation)

---

## 🔍 Key SQL Skills Used
- JOINs
- CTEs
- Window Functions (RANK, ROW_NUMBER)
- Aggregations
- CASE WHEN logic

---

## 📊 Insights
- Top 10% customers generate majority of revenue
- Certain products dominate sales performance
- Clear seasonal revenue patterns identified
